Code for the ICLR 2024 submission **Adaptive Bilevel Optimization**

The files `main.py` and `main_euclidean.py` can be used to run the RL experiments with negative entropy and squared norm regularizers.

The file `toy.py` has been used to produce Figure 1 in the pdf.

The file `adaptive.py` is the implementation of our algorithm in the benchmark for bilevel optimization available at https://github.com/benchopt/benchmark_bilevel
